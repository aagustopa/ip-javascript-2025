# Ejercicio Node para hacer tu propio comando "ls"

1. Crea un archivo llamado `myLs.js` y escribe un programa que nos permita ver la lista de archivos y carpetas de una carpeta especificada.
2. Que las carpetas se muestren de color verde y los archivos de color amarillo.
3. Que si yo solo ejecuto `node myLs.js` me muestre la lista de archivos y carpetas de la CARPETA ACTUAL.
4. Que si ejecuto `node myLs.js <ruta>` me muestre la lista de archivos y carpetas de la ruta especificada.
5. Que se muestren a la derecha de cada archivo (no de las carpetas) el tama√±o en bytes, KB, MB, GB.
