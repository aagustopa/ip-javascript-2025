# Chuleta npm


## ¡ANTES DE INSTALAR NADA POR PRIMERA VEZ!

1. Siempre que vayas a iniciar un proyecto en el que vas a instalar paquetes de terceros, deberíamos de crear antes un archivo `package.json`
2. Utiliza el comando `npm init -y`. Esto creará un archivo `package.json` con la configuración por defecto. 


## Instalar paquetes

```
npm install <paquete>

npm i <paquete>
```

## Instalar todos los paquetes cuando no tienes carpeta node_modules

```
npm install

npm i
```


## Desinstalar paquetes

```
npm uninstall <paquete>

npm un <paquete>
```
